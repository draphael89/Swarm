# Contributing

Thanks for your interest in Swarm.

## Before you start
Please open an issue first so we can discuss the change and confirm scope before any implementation work begins.

## Pull request policy
Unsolicited pull requests are not accepted. Pull requests without a linked, approved issue will be closed.

## Contribution flow
1. Open an issue.
2. Wait for maintainer approval.
3. Implement the agreed scope and open a PR linked to that issue.
